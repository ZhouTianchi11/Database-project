### Features of the project
1. Login Page
Unified login page for User and Super Admin
Role-based permission control and page redirection
2. Super Admin Features
Merchant Management
Merchant list displayed on the left
Add / delete merchants on the right
Product Catalog
Switch between different merchants via the dropdown menu
View all products under the selected merchant
Add products: enter name, price, stock, and three tags
Delete existing products
Order Management
View all orders submitted by users
3. User Features
Product List & Search
Display all product information
Search products by name or tag
Add products to shopping cart
Shopping Cart
Modify product quantity
Remove products
Checkout to place orders
My Orders
View all personal order records
Delete individual items in an order
Delete entire order records
User Account
View and manage personal account information
