🌟 E-Commerce Management System (Dual Role: User + Super Admin)
A complete e-commerce system with merchant, product, cart, and order management.
📖 Project Overview
This is a full-stack e-commerce management system supporting two roles: User and Super Admin. It implements merchant management, product management, shopping cart, order management, and personal account functions in a clean, easy-to-use interface.
🎯 Core Features
🔐 Login & Role Base Access
Unified Login Page: Supports login for both User and Super Admin.
Auto-Redirect: Automatically redirects to the corresponding dashboard based on user role.
🔑 Super Admin Module
Merchant Management: Left sidebar displays merchant list; Right panel supports adding/deleting merchants.
Product Catalog: Switch merchants via dropdown menu. View all products under the selected merchant.
Product Management: Add new products with name, price, stock, and 3 tags. Delete existing products.
Order Management: View a comprehensive list of all orders placed by users.
🧑‍💻 User Module
Product Browse & Search: Display all product information. Search products by name or tag.
Shopping Cart: Add products to cart. Update product quantity. Remove items from the cart.
Checkout Process: Place orders directly from the cart.
My Orders: View all personal order records. Remove single items from an order. Delete entire order records.
Account Info: View and manage personal account details.
